(2) 프리미어 리그 역대 및 현역 선수들의 전적을 평가하기 위한 핵심 지표(출전 경기 수, 골, 도움, 클린 시트, 승률 등)를 선정합니다.
(3) 홈페이지의 논리적인 메뉴 구조를 설계합니다. (예: 홈, 국가별 선수 탐색, 국적별 통계 랭킹, 역대 최고의 선수들, 커뮤니티 등)
(4) 메인 페이지(Home)의 화면 구성을 기획합니다. (인기 국가 섹션, 실시간 핫한 선수, 검색창, 리그 요약 정보 등)
(5) 국가별 상세 목록 페이지의 레이아웃을 설계합니다. (해당 국가 출신 선수 리스트, 포지션별 필터링, 국가 전체 통계 요약 등)
(6) 개별 선수 전적 상세 페이지의 화면 구성을 기획합니다. (선수 프로필, 시즌별 세부 기록, 커리어 하이라이트, 다른 선수와의 비교 차트 등)
(7) 데이터 시각화 요소(국가별 득점 분포 지도, 연도별 선수 진출 현황 그래프 등)를 제안하여 사용자 경험(UX)을 강화합니다.
(8) 위 조사 내용을 바탕으로 홈페이지 메뉴 설정 및 각 화면별 포함 요소가 상세히 담긴 웹사이트 기획안을 작성합니다.
